# Notas:

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Permite que se ingrese un número para realizar la tabla de     
                 multiplicar                                [número] [requerido]
  -l, --listar   Muesra la tabla de multiplicar      [booleano] [defecto: false]
  -h, --hasta    Permite indicar hasta qué número queremos la tabla
                                                        [número] [defecto: "10"]
```
